This skill delivers structured, scenario-grounded instruction across all 8 OTA domains AND the BIOL 2101 A&P Lab Final image set. Consistent three-layer architecture:

- 🔵 **Student Layer** — clinical concepts from zero, freshman-accessible, framework-driven
- 💼 **Executive Layer** — identical domain in business/operations language, no clinical jargon

Always run BOTH layers unless the user explicitly requests only one.

| Label | Structure | Function / Clinical Relevance |

| A | Head of femur | Articulates with acetabulum → hip joint (ball-and-socket) |
| B | Neck of femur | Common fracture site in falls; hip replacement site |

**OTA Hook:** Femoral neck fracture (B) → hip precautions → AT selection (sock aid, long-handled reacher, raised toilet seat). Greater trochanter (C) = palpation landmark for hip abduction ROM measurement.

### Image 2 — Meningeal Layers / Brain Cross-Section (Slide 14, labels A–K)

| Label | Structure | Function / Clinical Relevance |
|---|---|---|
| A | Skull (cranial bone) | Outer/inner compact tables + diploe (spongy bone) |
| B | Periosteum | Outer fibrous layer of skull |
| C | Dura mater (periosteal layer) | Tough outer meninx; adheres to inner skull surface |
| D | Dura mater (meningeal layer) | Inner dural layer; forms venous sinuses where layers separate |
| E | Arachnoid mater | Middle meninx; web-like, avascular |
| F | Subarachnoid space | Contains CSF; cushions brain against impact |
| G | Pia mater | Innermost meninx; directly contacts brain; follows every gyrus/sulcus |
| H | Epidural space | Between skull and dura; site of epidural hematoma |
| I | Subdural space | Between dura and arachnoid; site of subdural hematoma |
| J | Arachnoid granulations | Reabsorb CSF into venous sinuses → systemic circulation |
| K | Cerebral cortex (gray matter) | Gyri/sulci; higher cognition, motor, sensory processing |

**Mnemonic — meninges outside→in:** **D**ura → **A**rachnoid → **P**ia = **DAP**

**CSF pathway:** Choroid plexus → ventricles → subarachnoid space (F) → arachnoid granulations (J) → venous sinuses. Blockage = hydrocephalus.

**OTA Hook:** TBI disrupts cortex (K). Subdural hematoma (I) → sudden cognitive/behavioral change mid-session → stop, document, report to OT. Meningitis = inflamed DAP layers → medical emergency, not OT.

| Label | Structure | Origin → Insertion | Function | Plane |
|---|---|---|---|---|
| A | Gluteus medius | Ilium → Greater trochanter | Hip abduction + internal rotation | Frontal |

| D | Iliotibial band (IT band) | TFL/glute max → Gerdy's tubercle (tibia) | Force transmission; lateral knee stability | — |
| E | Rectus femoris | AIIS → Tibial tuberosity (via patella) | Hip flexion + knee extension (2-joint) | Sagittal |
| F | Vastus lateralis | Lateral femur → Tibial tuberosity | Knee extension | Sagittal |
| G | Biceps femoris (long head) | Ischial tuberosity → Fibular head | Knee flexion + hip extension | Sagittal |
| H | Biceps femoris (short head) | Lateral femur → Fibular head | Knee flexion only (1-joint) | Sagittal |
| I | Semimembranosus | Ischial tuberosity → Medial tibia | Knee flexion + hip extension + tibial IR | Sagittal |
| J | Semitendinosus | Ischial tuberosity → Medial tibia | Knee flexion + hip extension | Sagittal |
| K | Gastrocnemius (lateral head) | Lateral femoral condyle → Calcaneus | Plantarflexion + knee flexion (2-joint) | Sagittal |

**OTA Hook — ADL connections:**
- Gluteus medius (A) weak → Trendelenburg gait → fall risk → assess for mobility AT (D7)
- Rectus femoris (E) + vastus lateralis (F) = sit-to-stand power → PAPA-G grading target (D2)
- Hamstring tightness (G/I/J) → limits hip flexion → affects dressing, tub transfers, toilet hygiene
- Gastrocnemius spasticity (K) post-stroke → equinus → affects all ambulatory IADLs

| Label | Structure | Function | CN |
|---|---|---|---|
| A | Frontalis | Raises eyebrows; horizontal forehead wrinkles | VII |
| B | Orbicularis oculi | Closes eyelid; blinking | VII |
| C | Zygomaticus minor | Elevates upper lip | VII |
| D | Zygomaticus major | Draws mouth upward/lateral; smiling | VII |
| E | Orbicularis oris | Closes/protrudes lips; speech and eating | VII |
| F | Depressor anguli oris | Depresses corner of mouth; frowning | VII |
| G | Temporalis | Elevates mandible; mastication | V |
| H | Masseter | Strongest jaw closer; mastication | V |
| I | Buccinator | Compresses cheeks; keeps food between teeth | VII |
| J | Mentalis | Protrudes lower lip; wrinkles chin | VII |
| K | Platysma | Depresses mandible; tenses neck skin | VII |

**Key rule:** All facial expression muscles (A, B, C, D, E, F, J, K) = CN VII. Mastication muscles (G, H, I) = CN V.

**OTA Hook — oral motor/dysphagia:**
- Post-stroke facial droop = CN VII weakness → orbicularis oris (E) + buccinator (I) affected → food pocketing → aspiration risk
- Bell's palsy = CN VII damage → orbicularis oculi (B) cannot close → corneal exposure → OT intervention: eye protection, moisture drops, safety education
- Masseter (H) + temporalis (G) hypertonia → jaw clenching → limits oral hygiene ADL, feeding, communication

## SOAP Note Quick-Build Protocol

```
S — Subjective  : What the CLIENT said (quotes or paraphrase)
O — Objective   : What you MEASURED (ROM degrees, MMT grade, task steps completed, time)
A — Assessment  : What the data MEANS clinically (progress toward goal, barriers identified)
P — Plan        : What happens NEXT (next session focus, OT consult needed, HEP updates)
```

**Skilled service test:** Every note must answer — *"Could a family member have done this, or did it require OTA training?"* If you can't answer in 2 sentences, the note will be denied.

---

## SMART-S Goal Quick-Build Protocol

```
[Client] will [do what] [how well] [by when] [under what conditions]
+ Skilled justification: "requiring OTA instruction in [specific technique]"
```

**Red flags — reject and rewrite:**
- "Patient will improve ROM" → no function, no context, not billable
- "Patient will increase strength" → no occupation, no measure, no timeline
- "Patient will participate in therapy" → attendance, not outcome

## Exam Prep Mode

**BIOL 2101:** Name an image label → student identifies structure + states function. Cover all four embedded images + figures 8.3, 8.5, 14.5–14.6, 18.2.

**NBCOT:** Multiple choice scenario → case vignette → goal rewrite drill.

**High-yield NBCOT patterns:**
- OTA cannot independently evaluate or set goals
- Low-tech before high-tech in AT
- Autonomy vs. Beneficence = most common ethics tension
- Occupation-based > purposeful > preparatory (intervention hierarchy)
- Document skilled service or the claim gets denied
